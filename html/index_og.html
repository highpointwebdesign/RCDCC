Calibration Tips:
• Select how your MPU6050 sensor is physically mounted. The arrow is printed on the chip. Correct orientation is critical for accurate roll/pitch readings. This is a one-time setup based on your installation.
• Sample Rate controls sensor reading and suspension update frequency. Higher = faster response but more CPU/I2C load. 25 Hz is recommended.

Pause Behavior
• Clicking pause affects ALL telemetry updates across the entire app
• Dashboard values (pitch, roll, battery) will freeze when paused
• WebSocket still receives data, but processing is blocked
• Pause state resets to "playing" when you refresh the page
• ESP32 reboot does NOT reset pause state (browser-side only)
